Linear programming

This is short tutorial on linear programming (LP) that used to be part of the No bullshit guide to linear algebra, but was cut from the main text because it might not be interesting to all readers.

Business students will find this tutorial useful as it gives three points of view: - algorithmic (the steps of the simplex algorithm) - graphical (visual representation of row pivots) - numerical (fast computations using SymPy)
